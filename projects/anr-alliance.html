About this project

I did this project in my position as a Junior Web Producer in conjunction with the Restoration Team and WRI Brasil team. The website is a simple brochureware website that conveys the basics about assisted natural regeneration in English, Spanish, and Portuguese. The site uses colors and fonts inspired by a graphic from the Brasil team, which was put into a palette by the design team. I implemented these colors in a way that will ensure it is visually accessible to users. For this project, I helped with photo selection and placement, implemented site colors, held regular check-in meetings with the team, and incorporated custom CSS to the website to customize button styling, disable clicking categories and tags in the blog section, and set the active styling for the navigation.
